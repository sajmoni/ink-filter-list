## How to use

```tsx
<FilterList
  height={10}
  items={[
    {
      label: 'First',
      value: 'first',
    },
    {
      label: 'Second',
      value: 'second',
    },
  ]}
  onSubmit={(item) => {
    console.log('value', item.value)
  }}
/>
```

## :package: Install

```console
npm install ink-filter-list
```
